# Пошаговый запуск проекта (FastAPI + Telegram-бот)

> **ВАЖНО:** Сервер (FastAPI) и Telegram-бот должны быть запущены в РАЗНЫХ терминалах или вкладках! Если запускать оба процесса в одном терминале, бот не будет работать.

## 1. Освобождение порта 8000 (если занят)

Выполните в терминале:
```
lsof -i :8000
```
В выводе будет строка с PID (например, 12345). Завершите процесс:
```
kill -9 <PID>
```
Замените <PID> на номер процесса из вывода.

Если порт занят несколькими процессами (например, PID 69416 и 69418):
```
kill -9 69416 69418
```

---

## 2. Запуск FastAPI сервера

Перейдите в папку проекта и активируйте виртуальное окружение:
```
cd ~/Downloads/Notion\ and\ AI/notion-ai-web3-bot
source venv/bin/activate
```

Запустите сервер:
```
uvicorn main:app --reload
```

Если порт 8000 занят или хотите использовать другой порт:
```
uvicorn main:app --reload --port 8001
```

После запуска в терминале должно быть:
```
Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
```
или на другом порту.

Откройте сайт по адресу http://127.0.0.1:8000 (или http://127.0.0.1:8001)

---

## 3. Запуск Telegram-бота

Откройте НОВЫЙ терминал (или вкладку):
```
cd ~/Downloads/Notion\ and\ AI/notion-ai-web3-bot
source venv/bin/activate
python bot.py
```
или, если не работает:
```
python3 bot.py
```

---

## 4. Проверка работы
- В терминале не должно быть ошибок.
- В Telegram напишите вашему боту `/start` — он должен ответить приветствием, картинкой и кнопками.
- Если бот не реагирует или появляются ошибки — скопируйте их сюда для разбора.
